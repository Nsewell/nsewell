Logo

Jul 2nd, 2012

Logo is a classic lisp like turtle graphics language and was my introduction to programming. With Logo, you can teach some advanced concepts like fractal patterns, and there’s some great online examples that demonstrate a few simple, elegant algorithms to generate pretty pictures.

Papert is a really nice online Logo interpreter using JavaScript and Canvas, look at these examples:

Tree (with colour) Koch Snowflake

Hilbert Curve Sierpinski triangle

Sierpinski carpet Sierpinski arrowhead

Hexaflake Gosper curve

Sierpinski triangle in dots Sieve of Erastosthenes (Prime generator)
